# 英雄坛说·100%内容复刻框架（移动端）

> 本版本按你的要求继续扩展，目标对齐《英雄坛说》全量玩法内容（以可迭代框架 + 已落地主线模块方式实现）。

## 快速开始

```bash
python3 -m http.server 8080
```

- 电脑：`http://localhost:8080`
- 手机（同 Wi-Fi）：`http://你的电脑IP:8080`

## 现在可玩的完整流程

1. 平安镇找里正接任务
2. 去黑风岭击败山贼头目
3. 回镇后可练功、强化、赌博、进黑市
4. 选择加入少林或武当并学门派武学
5. 提升声望触发婚配事件
6. 通过存档/读档持续推进
